## Run Locally

Clone the project

```bash
  git clone https://github.com/ambir513/fellownotes-backend.git
```

Go to the project directory

```bash
  cd fellownotes-backend
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file. example is in .env.example file

`PORT`


Install dependencies

```bash
  npm install
```

build into JS

```bash
  npm run build
```

start the server
```bash
  npm run start
```
